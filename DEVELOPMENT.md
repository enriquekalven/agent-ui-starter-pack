# Development

## Project Structure

- `src/a2ui`: Core rendering logic for A2UI components.
- `src/components`: UI components like Home and ThemeToggle.
- `src/docs`: Documentation layout and pages.
- `src/index.css`: Main styling with CopilotKit aesthetic.

## Adding Components

You can add new A2UI components by modifying the `A2UIRenderer.tsx` file and adding mapping for new component types.
